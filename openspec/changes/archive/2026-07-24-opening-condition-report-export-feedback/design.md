## Context

报告页通过后端导出 DOCX。导出请求是异步的，浏览器可能阻止请求完成后的 `window.open`，因此必须把结果保留在前端状态并在当前页面渲染。

## Decisions

1. `onExportReport` 接收明确的 `taskId`，由报告组件传入当前选中的任务。
2. 前端保存短生命周期的导出状态和下载 URL，不写入平台任务快照。
3. 成功和失败状态使用报告页已有的记录展示容器，保持当前页面结构稳定。

## Non-Goals

- 不把预签名下载 URL 持久化到后端。
- 不重构报告页整体视觉布局。
- 不改变 HTTP tools adapter 和导出 API。
